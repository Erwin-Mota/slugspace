# SlugSpace

A community platform for UCSC students to connect, find study groups, discover campus clubs, and get matched with residential colleges.

## Overview

SlugSpace helps UCSC students:
- **Find study groups** - Search through 1000+ UCSC courses and join study groups for your classes
- **Discover clubs** - Browse campus clubs organized by category with personalized recommendations
- **College matching** - Take a survey to get matched with one of UCSC's 10 residential colleges
- **Build your profile** - Set up your interests and preferences to get better recommendations

The platform features a modern, responsive UI built with UCSC's official blue and yellow color scheme.

## Tech Stack

Next.js, TypeScript, Tailwind CSS, NextAuth.js, PostgreSQL, Prisma
